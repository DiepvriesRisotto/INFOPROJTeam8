Leitner system

To us the Leitner system you first need to create flashcards that feature topics and material that you are studying. Secondly, you need 3 to 5 boxes to house your flashcard. Finally, you need to label each box with a designated study time period. All your cards should start in box 1. If you answer a card correctly, move it to box 2. If you answer incorrectly, move it back to box 1. Simply put, if you get a card wrong, move it to the previous box, if you get a card correct, move it to the next box. Once you are done going through all the cards, you now have a system to distinguish which cards need to be reviewed more often.
